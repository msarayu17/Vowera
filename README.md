# 💍 Wedding Planner Website - VOWERA

A responsive and elegant wedding planning website designed to help users explore wedding services, packages, galleries, and event information in one place.

## ✨ Features

* 🏠 Attractive Home Page
* 📸 Wedding Gallery
* 🎉 Service Information
* 💎 Wedding Packages
* 👥 Team Section
* 📝 Registration Form
* 📞 Contact Page
* ✅ Thank You Page
* 🔒 Privacy Policy
* 📜 Terms and Conditions
* 📱 Responsive Design
* 🎨 Modern User Interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/WeddingPlanner.git
```

### Open the Project

Navigate to the project folder:

```bash
cd WeddingPlanner
```

Open `home.html` in your browser to view the website.
